# Deploying to Node.js Server

Rasengan.js is a framework for building web applications using React. It is designed to be used with Node.js.

<br />
In this guide, you will learn how to deploy a Rasengan.js app to a Node Server.

## Deploy 

To deploy your Rasengan.js app to a Node.js server, you need to configure your project to use the `@rasenganjs/serve` adapter.
Have a look at the <Link to="/packages/serve">`Node adapter documentation`</Link> for more information.
