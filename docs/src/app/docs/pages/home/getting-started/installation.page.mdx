## Pre-requisites

- <a href="https://nodejs.org" target="_blank" rel="noopener noreferrer">Node.js 20.18</a> or later.
- macOS, Windows, and Linux are supported.

## Installation

### Automatic Installation

The best way to start a new **Rasengan.js** project is with **[create-rasengan CLI](/docs/api-reference/create-rasengan-cli)**, which sets everything up for you automatically. 
    
Just follow the following steps to get started:

<StepItem
  step="01"
  title="Create a new project"
  content={`
    Run the following command to create a new project.
  `}
  className='mt-8'
>
```bash title="Terminal"
npx create-rasengan@latest
```
</StepItem>


<StepItem
  step="02"
  title="Follow the prompts"
  content={`
    On the installation process, you will see the following prompts:
  `}
>
```bash title="Terminal"
- What would you like to name your project?
- Which language would you like to use for your project?
- Which template would you like to use?
```
</StepItem>

<StepItem
  step="03"
  title="Install dependencies"
  content={`
    After the prompts, \`create-rasengan\` will create a folder with your project name. Move yourself inside and install the required dependencies.
  `}
>
```bash title="Terminal"
cd my-app
npm install
```
</StepItem>

<StepItem
  step="04"
  title="Run the application"
  content={`
    Run the following command to start the development server.
  `}
>
```bash title="Terminal"
npm run dev
```
</StepItem>

The following steps will guide you through setting up a new **Rasengan.js** project manually using `JavaScript`.

<StepItem
  step="01"
  title="Install dependencies"
  content={`
    Create a new folder first and move yourself inside.

    Then generate a new \`package.json\` file and install the required dependencies.
  `}
>
```bash title="Terminal"
mkdir my-app
cd my-app
npm init -y
npm install rasengan@latest @rasenganjs/serve@latest react@latest react-dom@latest
npm install --save-dev cross-env vite@latest
```
</StepItem>

<StepItem
  step="02"
  title="Add scripts into package.json"
  content={`
    Open the \`package.json\` file and add the following scripts.
  `}
>
```json title="package.json"
"scripts": {
  "dev": "rasengan dev",
  "build": "rasengan build",
  "serve": "rasengan-serve ./dist"
}
```
</StepItem>

<StepItem
  step="03"
  title="Create root directories and files"
  content={`
    First, create the necessary directories at the root of your project.
    
    - \`public\`: Contains static assets like icons and images.
    - \`src\`: Contains your applicationâ€™s source code.
    
    Then, create the \`rasengan.config.js\` configuration file.
  `}
>
```bash title="Terminal"
mkdir public src
touch rasengan.config.js
```
</StepItem>

<StepItem 
  step="04" 
  title="Configure Rasengan.js" 
  content={`
    Open the \`rasengan.config.js\` file and add the following configuration. 
  `}
>
```javascript title="rasengan.config.js" showLineNumbers
import { defineConfig } from "rasengan";

export default defineConfig({});
```
</StepItem>

<StepItem
  step="05"
  title="Create sub-directories and files"
  content={`
    Inside the \`src\` directory, create an \`app\` folder to store your application's pages.

    Then create a \`home.page.jsx\` and \`app.router.js\` files inside the \`app\` directory.

    Finally create a \`main.jsx\`, \`template.jsx\` and \`index.js\` files inside the \`src\` directory.
  `}
>
```bash title="Terminal"
mkdir src/app

touch src/app/home.page.jsx src/app/app.router.js

touch src/main.jsx src/template.jsx src/index.js
```
</StepItem>

<StepItem
  step="06"
  title="Create a simple page"
  content={`
    Open the \`home.page.jsx\` file and add the following code to create a simple page.
  `}
>
```jsx title="home.page.jsx" showLineNumbers
const Home = () => {
  return (
    <main>Home Page</main>
  )
}

Home.path = "/";
Home.metadata = {
  title: "Home",
  description: "Home page",
}

export default Home;
```
</StepItem>

<StepItem
  step="07"
  title="Create a router"
  content={`
    Open the \`app.router.js\` file and add the following code to create a router.
  `}
>
```javascript title="app.router.js" showLineNumbers
import { RouterComponent, defineRouter } from "rasengan";
import Home from "@/app/home.page";

class AppRouter extends RouterComponent {}

export default defineRouter({
  pages: [Home]
})(AppRouter);
```
</StepItem>

<StepItem
  step="08"
  title="Define the main component"
  content={`
    Open the \`main.jsx\` file and add the following code to define the main component.
  `}
>
```jsx title="main.jsx" showLineNumbers 
import AppRouter from "@/app/app.router";

export default function App({ Component, children }) {
  return <Component router={AppRouter}>{children}</Component>;
}
```
</StepItem>

<StepItem
  step="9"
  title="Define the template HTML"
  content={`
    Open the \`template.jsx\` file and add the following code to define the template HTML.
  `}
>
```jsx title="template.jsx" showLineNumbers
export default function Template({
  Head,
  Body,
  Script
}) {
  return (
    <html lang="en">
      <Head>
        <meta charSet="UTF-8" />
        <link rel="icon" type="image/svg+xml" href="/rasengan.svg" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      </Head>

      <Body>
        <Script />
      </Body>
    </html>
  );
}
```
</StepItem>

<StepItem
  step="10"
  title="Define the entry point"
  content={`
    Open the \`index.js\` file and add the following code to define the entry point.
  `}
>
```jsx title="index.js" showLineNumbers {4}
import { renderApp } from 'rasengan/client';
import App from './main';

renderApp(App, { reactStrictMode: true });
```
</StepItem>

<StepItem
  step="11"
  title="Run the application"
  content={`
    Run the following command to start the development server.
  `}
>
```bash title="Terminal"
npm run dev
```
</StepItem>
</Tabs.Item>
