# Introduction to Rasengan.js

Rasengan.js is a modern, fast and extensible framework for building web applications.

## Features

- File-based routing
- CSS Support
- SSR / CSR
- MDX Support

## Installation

```bash title="Terminal"
npm install rasengan
```

## Usage

```tsx
import { defineRouter } from 'rasengan';

const router = defineRouter({
  pages: [],
});

export default router;
```

## Example

```tsx
import { defineRouter } from 'rasengan';

const router = defineRouter({
  pages: [
    {
      path: '/',
      component: Home,
    },
  ],
});

export default router;
```
